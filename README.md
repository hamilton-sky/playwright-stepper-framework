# automation-stepper

Automation suite for OpenLibrary with Stepper-based flows, auth, interfaces,
and performance checks.

## Overview
- End-to-end and API validation for OpenLibrary flows
- Auth and interface coverage
- Performance checks and reporting

## Structure
- `openlibrary/`: OpenLibrary automation code and tests
- `stepper/`: Stepper config and related assets

## Quick Start
1. Create a virtual environment and install dependencies (if applicable).
2. Run tests with `pytest`.

## Notes
- Reports and artifacts are generated under `openlibrary/` subfolders.
